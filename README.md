# 🎬 Movie Terminal (v4.2)

**Movie Terminal** is a high-fidelity terminal emulator script designed for use as a background prop in film and video production. It simulates complex system activity, network handshakes, and data decryption sequences with a highly customizable "cyber-ops" aesthetic.

## 🚀 Features
- **Real-World Logic:** Uses the `wonderwords` engine to generate human-readable logs rather than pure gibberish.
- **Dynamic ASCII Banners:** Large-scale block letter headers for system transitions.
- **Custom Themes:** 7 built-in color schemes including Matrix Green, Cyberpunk Amber, and Crimson Alert.
- **Director Controls:** Adjustable intensity for scroll speed, banner frequency, and progress bar interruptions.
- **Responsive:** Automatically detects terminal width for a full-screen "wall of data" look.
